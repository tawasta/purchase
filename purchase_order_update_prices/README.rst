============================
Update Purchase Order prices
============================

Price computations are done by core's functions.

Configuration
=============
Set the Scheduled action to active, if the computation needs to happen daily.

Usage
=====
Go to Settings --> Scheduled Actions and search for
Update Purchase Order line prices action and press Run Manually or
set that action as active.
