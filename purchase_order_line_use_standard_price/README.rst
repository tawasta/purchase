===========================================================
Purchase Order: Use standard price as default PO line price
===========================================================

* Use standard price as line price, if no supplier price is set

Configuration
=============
* None needed

Usage
=====
* Create a new PO with a line whose product does not have a supplier price.
  The product's standard price is suggested automatically on the PO line.
