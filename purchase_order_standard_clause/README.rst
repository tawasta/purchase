==============================
Purchase Order Standard Clause
==============================

* Adds a company-specific standard clause to Purchase Orders

Configuration
=============
* Configure (and translate) the standard clause in Purchases -> Configuration -> Settings
* The clause appears as a default for Purchase Orders when a vendor is selected, but the text can be edited if necessary
